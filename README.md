## 📦 安装

```bash
# 使用 npm
npm install weavefox

# 使用 yarn
yarn add weavefox

# 使用 pnpm
pnpm add weavefox
```

## 🎯 快速开始

```javascript
import weavefox from 'weavefox';

// 开始使用 WeaveFox
const app = weavefox.create({
  // 你的配置
});

app.run();
```
